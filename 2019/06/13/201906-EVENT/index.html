回應偉大的召喚

六月 13, 2019 優惠活動

社群媒體、政治談話、職場競爭…，有許多的不友善在生活中蔓延，並讓我們付出了無數的代價。好消息是，有些簡單的方法能作為愛的容器、填補內心的空洞。

試著深呼吸、放下一切，看看我們能做些什麼？我們知道可以透過改變內在世界和改變對世界的態度，來創造一個新的世界。

如果你想經驗和平，為另一個人提供和平。－達賴喇嘛

作為一個接收的容器，當周遭充滿負面的能量和情緒時，它將會挑戰我們的健康和幸福，而當你在做的是一件讓你感到熱忱、活力和歸屬感的事情，或者享受和你喜愛的親友們的陪伴、為你所愛的人提供溫暖的擁抱。

這些愛、付出、滿足與成就感會讓大腦釋放更多的多巴胺，提升5-羥色胺的分泌，而5-羥色胺對於心血管、內分泌和情緒的提升是至關重要的。對於腦部的影響，包括情緒、行為和壓力正是精油和花精擅長的領域，這裡有些複方能帶來振奮、鼓舞和積極的影響：

內在小孩

當需要感受到更多的愛、寧靜與撫慰，克服眼前的各種挑戰時，這配方能提供協助。

  • 葡萄柚精油
  • 綠薄荷精油
  • 柑橘精油
  • 報春花花精

你可以使用喜愛的擴香儀、水氧機或滴入個人芳香吸入器使用，也可以直接將花精滴入到水中飲用。

神聖空間

創造一個清晰、積極和受保護的個人空間是必要的，在這空間當中你可以舒適、真實的做你自己，並在與他人互動時保持愉快。

  • 佛手柑精油
  • 杜松漿果精油
  • 迷迭香精油
  • 朱草花精

你可以將這配方用植物酒精做成空間噴霧，或滴到芳香項鍊來配戴使用，也可以直接將花精滴入到水中飲用。

充滿希望

生命中難免有無助、無力或缺乏活力的時刻，在面對慢性疾病、長期困難或毫無選擇時，你需要更多的歡樂、希望和信心。

  • 芫荽精油
  • 波旁天竺葵精油
  • 茉莉原精
  • 荊豆花精
  • 龍膽花精

你可以滴入到芳香吸入器或擴香儀中使用，也可以添加到荷荷巴油稀釋後塗抹在腹部、肩頸，也可以直接將花精滴入到水中飲用。

獲得指引

將自我意識與更高意識相連結，以獲得更高的指引、回應偉大的召喚，讓你擁有突破現狀的內在力量。

  • 阿拉伯乳香精油
  • 沒藥精油
  • 欖香脂精油
  • 艾菊花精

你可以在需要獲得指引、連結或找到方向感時，稀釋後塗抹在頭部或頸部，或者將花精滴入水中飲用。

透過簡單的步驟，你能夠在很短的時間創造愉快的氛圍、感受到更多的愛，使身心獲得平衡。同時，我們也可以為周遭的人帶來更多的快樂。因為「你不能從空的杯子倒出任何東西」，
先回應內在的呼喚，然後分享給更多的人們。
